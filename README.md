# 🛒 ShopEase — Checkout Optimization

## Overview
ShopEase is a product case study focused on diagnosing and resolving
high cart abandonment in an e-commerce checkout flow.
This project covers end-to-end product thinking — from problem
discovery to SQL data modeling, dashboard analytics, and a
complete PRD with user stories.

## 🔍 Problem Statement
72% cart abandonment rate over 3 months.
Customers adding items but dropping off before payment.

## 🎯 Business Goals
- Reduce abandonment rate: 72% → 55%
- Improve conversion rate: 65% → 75%
- Increase payment success rate

## 📊 Key Metrics (from Data Analysis)
- Payment Success Rate: 50%
- Payment Failure Rate: 16.67%
- Credit Card Failure Rate: 50%
- Net Banking Pending Rate: 100%
